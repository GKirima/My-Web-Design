Project 1: Deploying a VPC & EC2 Instance via AWS CLI

This project demonstrates how to provision a Virtual Private Cloud (VPC) from scratch and launch an EC2 instance using AWS Command Line Interface. It includes subnets, routing tables, security groups, and SSH access.

Project 2: Deploying a Secure 3-Tier Architecture in AWS

A practical implementation of a three-tier web architecture using Amazon VPC, EC2, RDS, and Load Balancer. This project focuses on high availability, network isolation, and secure access between tiers.

Project 3: Migrating a Legacy Architecture to AWS

This project outlines a step-by-step process for analyzing, planning, and migrating legacy infrastructure to the AWS Cloud. It covers rehosting (lift-and-shift), storage migration, and network configuration for minimal downtime.
